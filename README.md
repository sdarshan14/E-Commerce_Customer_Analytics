# Ecommerce Customer Analytics

## Overview

This project analyzes e-commerce transaction data to understand customer purchasing behavior, product performance, revenue trends, and sales patterns. The objective is to generate business insights through exploratory data analysis and visualization techniques.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL
- Jupyter Notebook

## Dataset

Dataset: Online Retail Dataset

The dataset contains transaction-level information including:

- Invoice Number
- Product Description
- Quantity
- Invoice Date
- Unit Price
- Customer ID
- Country
